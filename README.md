# nrperry.com

Personal portfolio for Nathan (Nate) Perry. Built with Next.js 14, Tailwind CSS,
Framer Motion, and deployed on Vercel.

## Develop

```bash
npm install
npm run dev
```

App runs at http://localhost:3000.

## Build

```bash
npm run build
npm start
```

## Deploy

Auto-deployed to Vercel on pushes to `main`. The `CNAME` file is no longer
needed — the custom domain is configured through Vercel project settings.
